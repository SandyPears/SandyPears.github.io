Tax Calculator

Summary

The Employee Payroll Calculator is a simple UI-based system for managing employee payroll efficiently with accurate tax calculations.

Tech and Features

  • Employee Selection & Input: Users select an employee and enter weekly hours.
  • Automated Calculations: The system calculates gross pay, tax deductions based on thresholds, superannuation, and net pay.
  • Compliance: Ensures accurate tax and superannuation calculations in line with Tax Office regulations.
  • Data Management: Users can save payroll details to a CSV file for record-keeping.

Point of Sale App

Summary

The Bean Scene Ordering App was a collaborative project that is a mobile application designed to streamline ordering, ticket management, and menu browsing for a restaurant. It features role-based navigation, allowing managers, servers, and chefs to access relevant functionalities.

Tech and Features

  • Login & Registration: Secure staff login and role-based access.
  • Home Page: Central hub for navigation based on user roles.
  • Menu Page: Browse and download the menu as a PDF.
  • Ordering System: Filter menu items, add to an order, specify a table, and create tickets.
  • Ticket Management: Servers and kitchen staff can view, complete, and delete tickets.
  • Admin Features: Manage menu items, add/edit dishes with details, and filter/search the menu.

Reservation Website

Summary

This collaborative project is designed to simplify reservation management, menu browsing, event inquiries, and provide information on the restaurant's services, while enabling staff and admin to efficiently handle reservations, seating, and menu updates.

Tech and Features

  • Reservation Booking: Choose the date, time, party size, preferred seating area, contact details, and any specific requests for a reservation.
  • Login & Registration: Secure staff login with role-based access for different functionalities.
  • Menu Page: Browse and download the menu as a PDF for easy reference.
  • Events: View event details and learn how to inquire about booking events.
  • Staff & Admin Features: Manage sittings and reservations, view, confirm, and manage both past and upcoming reservations, and add, edit, or delete seating times and types.
